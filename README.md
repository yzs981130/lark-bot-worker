## Config

Environment variables:

```sh
CLOUDFLARE_ACCOUNT_ID=xxx
CF_API_TOKEN=xxx
# (Optional)
CF_ZONE_ID=xxx

# Lark/Feishu bot webhook
WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxx
# (Optional) Lark/Feishu bot signature secret
SIGN_SECRET=xxxxxxxxxxxx
```
